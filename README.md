# Java Quiz Application

A simple desktop-based **Quiz Application** developed using **Java Swing and AWT**.
The application presents multiple-choice questions with a timer and calculates the user's score based on correct answers.

---

## Features

* Graphical User Interface (GUI) using Java Swing
* Multiple-choice questions with four options
* Timer for each question
* Automatic answer validation
* Final score and percentage calculation

---

## Technologies Used

* Java
* Java Swing
* AWT
* Event Handling

---

## How to Run the Project

1. Clone the repository

git clone https://github.com/your-username/java-quiz-application.git

2. Navigate to the project folder

cd java-quiz-application

3. Compile the program

javac Quiz.java

4. Run the program

java Quiz
